Reeborg

Reeborg 原始碼: https://github.com/mdecycu/reeborg

從 world.html 可以發現使用 Brython 3.2.4, 釋出時間 2016.01, 而 2015.06 為 ECMAScript 6 釋出時間, 因此判定 Reeborg 所使用的 Javascript 語法大多為 ECMAScript version 5.

執行 Reeborg 最簡單的方式:

  1. 啟動可攜程式環境
  2. cd tmp
  3. git clone https://github.com/mdecycu/reeborg.git
  4. cd reeborg
  5. python -m http.server
  6. 然後使用瀏覽器連結至 http://localhost:8000/world.html

On world.html, 設定選擇 Editor:

    <select id="python_choices" title="Select programming mode">
        <option value="blockly">Blockly</option>
        <option value="editor" selected>Editor</option>
        <option value="repl">Interpreter</option>
